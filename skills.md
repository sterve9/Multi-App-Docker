# 🎯 OBJECTIF GLOBAL

Construire des systèmes digitaux qui génèrent des leads et des ventes automatiquement via WhatsApp pour des infopreneurs en Afrique francophone.

Priorité absolue : CONVERSION > DESIGN > COMPLEXITÉ

---

# 🥇 CORE SKILLS (PRIORITAIRES)

## 1. Landing Pages (ULTRA CONVERSION)
- HTML5 (léger, rapide)
- Tailwind CSS (design rapide)
- JavaScript vanilla (interactions simples)
- Next.js (si nécessaire uniquement)

Objectif :
- Chargement rapide (<2s)
- Mobile-first (90% trafic mobile)
- Bouton WhatsApp visible immédiatement
- Structure courte (scroll minimal)

---

## 2. Automatisation (CŒUR DU BUSINESS)
- n8n (auto-hébergé)
- Webhooks
- API HTTP
- Green API (WhatsApp)
- Brevo SMTP (emails)

Objectif :
- Répondre automatiquement aux leads
- Envoyer messages WhatsApp instantanés
- Relancer automatiquement
- Gérer séquences post-achat

---

## 3. Conversion & Copywriting
- Hooks émotionnels
- Structure AIDA (Attention, Intérêt, Désir, Action)
- Texte simple, clair, direct
- CTA orienté WhatsApp

---

# 🥈 SECONDARY SKILLS

## Backend
- Python (FastAPI)
- API REST
- PostgreSQL

## Frontend avancé
- Next.js
- TypeScript
- Zustand (state)

# 🥉 CONTENU (OPTIONNEL)

# 🏗️ INFRASTRUCTURE
- Docker / Docker Compose

# 📱 CONTRAINTES AFRIQUE (CRITIQUES)

- Connexion lente (3G fréquente)
- Téléphones bas de gamme
- Utilisation massive de WhatsApp

OBLIGATOIRE :
- Pages légères
- Images compressées (webp)
- Zéro scripts inutiles
- Temps de chargement rapide

---

# 🚫 INTERDICTIONS

- Pas de 3D lourde (Three.js complet interdit)
- Pas de librairies inutiles
- Pas de design complexe qui ralentit
- Pas de dépendances inutiles

---

# 🚀 OBJECTIF FINAL

Créer des systèmes qui :

1. Attirent (landing page)
2. Capturent (WhatsApp)
3. Convertissent (automation n8n)

---
